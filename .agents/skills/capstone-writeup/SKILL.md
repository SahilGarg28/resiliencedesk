# Skill: capstone-writeup

Use this skill when preparing and finalizing the submission documentation or the final capstone writeup for ResilienceDesk.

## Guidelines

1. **System & Topology Architecture**:
   - Provide a clear layout diagram of the agent topology (e.g. Architect, Java-Engineer, Security-Reviewer, QA-Eval, LLM interaction flows).
   - Describe tools and MCP configurations.

2. **Security & Safety Guardrails**:
   - Detail the safety guardrails, disclaimer implementation, and threat modeling outcomes.
   - Explain how prompt injections and other risks were mitigated.

3. **Evaluation Results**:
   - Summarize findings from the `run-eval` test suites.
   - Present grading matrix, key outcomes, and agent performance metrics.

4. **Kaggle Submission Format**:
   - Ensure the writeup conforms to all Kaggle x Google AI Agents course requirements.
   - Package all assets and screenshots neatly.
